教材管理系统。

大体思路为首页-》登陆界面-》进入身份对应的网页-》点击退出-》首页
共有3种身份，管理员，教师，班长

管理员具有的功能：
    
    - 查看所有教材
    - 更改教材库存
    - 增加新库存教材
    - 删除库存教材
    - 显示所有订单
    - 更新订单状态
    - 显示所有教师/学生
    - 编辑/增加/删除 教师/学生
    
教师具有的功能：

    - 查看所有教材
    - 查看个人订单
    - 增加个人订单
    - 编辑个人信息
      
班长具有的功能：

    - 查看本班订单
    - 编辑个人信息
